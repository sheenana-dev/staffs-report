# KPIs Per Department

## Purpose
Define what success looks like for each department with measurable numbers. KPIs (Key Performance Indicators) tell you whether a department is healthy, struggling, or thriving — without guessing.

## When
- Set KPIs at the start of each quarter
- Review monthly during the monthly review meeting
- Adjust only if business conditions fundamentally change (not because targets are hard)

## The B-Ticket KPI Framework

### Company-Level KPIs (The Big Picture)
These are the numbers that tell you if B-Ticket is winning overall:

| KPI | What It Measures | Target |
|-----|-----------------|--------|
| Total active merchants | Supply side health | Set quarterly |
| Monthly app downloads | Demand side growth | Set quarterly |
| Coupon redemption rate | Marketplace health | 30%+ of claimed coupons |
| Subscription revenue | Core revenue stream | Set quarterly |
| Banner ad revenue | Secondary revenue stream | Set quarterly |
| Active subscribers | Paying user base | Set quarterly |
| Merchant churn rate | Retention | Below 5% per month |

Every department KPI should connect back to at least one of these.

## How to Use KPIs

### Step 1: Set Targets Quarterly
1. Review last quarter's actual numbers
2. Set targets that are challenging but achievable (10-20% improvement is healthy)
3. Get buy-in from department leads — they need to believe the target is fair
4. Document targets in the quarterly planning doc

### Step 2: Track Monthly
1. Department leads report their KPIs in the monthly review
2. Use a simple red/yellow/green status:
   - Green: On track or above target
   - Yellow: Within 10% of target — needs attention
   - Red: More than 10% below target — needs action plan

### Step 3: Act on Red KPIs
1. Don't panic — one red month isn't a crisis
2. Ask: "What changed? What's causing this?"
3. Create a specific action plan with the department lead
4. Review progress in 2 weeks, not next month
5. If red for 2+ months, the target or the approach needs to change

### Step 4: Celebrate Green KPIs
1. Call out wins in the Monday sync
2. Share successes across departments
3. If a KPI is consistently green, consider raising the target

## Red Flags
- A department has no measurable KPIs — they can't improve what they don't measure
- All KPIs are green every month — targets are too easy
- KPIs change every quarter — you're chasing shiny objects instead of building momentum
- Department leads don't know their own KPIs — the metrics aren't embedded in daily work
- KPIs measure effort (hours worked) instead of outcomes (results delivered)
- One department is always red while others are green — might be a resource or leadership issue
